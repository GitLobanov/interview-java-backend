---
Определение: Представляет операцию, принимающую один аргумент типа T и возвращающую результат того же типа T.
Примечание: Реализован на основе Function, но с одним типом данных
---
### Методы

```
apply(T t): Метод, который принимает один аргумент типа T и возвращает результат типа T.
```
