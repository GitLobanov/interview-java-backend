Определение: Представляет функцию, принимающую два аргумента типа T и U и возвращающую результат типа R. Этот интерфейс используется для выполнения операций, которые требуют двух входных значений и возвращают один результат.
Примечание:
---
### Методы

```
apply(T t, U u): Метод, который принимает два аргумента типа T и U, а затем возвращает результат типа R.
andThen(Function<? super R, ? extends V> after): Возвращает составную функцию, которая сначала применяет текущую BiFunction к двум аргументам, а затем применяет функцию after к результату.
```
