---
Определение: Представляет собой операцию, принимающую один входной аргумент типа T и не возвращающую результата
Примечание:
---
### Методы

```
accept(T t): Метод принимает один аргумент типа T и выполняет над ним какое-то действие.
andThen(Consumer<? super T> after): Возвращает составной Consumer, который сначала выполняет действие текущего Consumer, а затем действие, переданное в andThen.
```
